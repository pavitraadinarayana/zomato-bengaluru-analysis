# zomato-bengaluru-analysis
Zomato Bengaluru Restaurant Analysis using Python, SQL and Tableau.

## Overview
Analysed 9,005 restaurants across Bengaluru to find 
cuisine trends, rating patterns and delivery insights.

## Key Findings
- Whitefield has the most restaurants (563)
- North Indian is the most popular cuisine
- Luxury restaurants (₹1000+) rate highest at 4.03
- 61% of restaurants offer online delivery

## Tools Used
Excel → Python (Pandas) → SQL (SQLite) → Tableau

## Files
- `zomato.ipynb` — Full Python analysis notebook
- `zomato_clean.csv` — Cleaned dataset (9,005 rows)
- `top_locations.png` — Top 10 areas chart
- `rating_distribution.png` — Rating distribution chart
- `online_vs_rating.png` — Delivery vs rating chart
